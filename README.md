# post-web

## Project setup with Docker
```
git clone https://github.com/ervinsvilumsons/post-web.git
docker-compose up -d --build

http://localhost:9002
```

## Project setup without Docker
```
git clone https://github.com/ervinsvilumsons/post-web.git
npm install
npm run serve

http://localhost:8080
```
